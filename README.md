# 📄 PDF to Audio Speech Converter

## 📌 Overview
This project converts PDF files into audio speech using Text-to-Speech (TTS) technology. It helps users listen to documents instead of reading them.

---

## 🎯 Features
- Convert PDF text into speech  
- Supports multiple pages  
- Save output as audio file (MP3/WAV)  
- Easy to use interface  
- Useful for visually impaired users  

---

## 🛠️ Technologies Used
- Python  
- PyPDF2 / pdfplumber  
- pyttsx3 / gTTS  
- playsound  

---

## ⚙️ How It Works
1. Load PDF file  
2. Extract text from pages  
3. Process text  
4. Convert text into speech  
5. Save/play audio
